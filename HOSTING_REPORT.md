# Обоснование выбора хостинга для проекта Voguerate

## 1. Frontend (React SPA)
**Выбор:** [Vercel](https://vercel.com/) или [Netlify].
- **Причины:**
    - **CDN:** Автоматическое распределение контента по всему миру. Минимальный TTFB (время до первого байта).
    - **SSL:** Автоматическая генерация бесплатных сертификатов Let's Encrypt.
    - **CI/CD:** Деплой из GitHub в один клик при каждом пуше.
    - **Edge Functions:** Возможность ускорить API-запросы через проксирование.

## 2. Backend (Laravel API + PostgreSQL)
**Выбор:** [Timeweb Cloud](https://timeweb.cloud/) (или Selectel).
- **Причины:**
    - **Локация:** Серверы в РФ (Санкт-Петербург/Москва) обеспечивают минимальный пинг для целевой аудитории.
    - **Docker Support:** Легкий деплой через наш существующий `docker-compose.yml`.
    - **Базы данных:** Возможность аренды выделенной БД PostgreSQL с автоматическими бэкапами.
    - **Цена:** Оптимальное соотношение ресурсов (CPU/RAM) для учебного/дипломного проекта.

## 3. Проверка работоспособности базы данных
- [x] Миграции успешно протестированы локально.
- [x] Используется Eloquent ORM для защиты от SQL-инъекций.
- [x] Для оптимизации запросов внедрена жадная загрузка `with()` (в контроллерах Laravel).
