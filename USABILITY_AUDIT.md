# Юзабилити-аудит и анализ поведения (Пр.№5)

## 1. Поведенческие метрики
- **Целевой показатель отказов (Bounce Rate):** < 40% (за счет уникального дизайна в стиле Techwear).
- **Глубина просмотра:** 4-5 страниц за сессию (связки Главная -> Дропы -> Детали предмета -> Профиль дизайнера).
- **LTV (Lifetime Value):** Возврат пользователей через подписку на уведомления о новых дропах.

## 2. Юзабилити-фишки
- **Sidebar Navigation:** Удобный доступ ко всем разделам (88px ширина), не перекрывает контент.
- **Glassmorphism:** Эстетическая привлекательность, повышающая лояльность.
- **Mobile First:** Интерфейс полностью адаптирован под большие пальцы (кнопки > 44px).

## 3. Социальные факторы
- Кнопки шеринга в соцсети для каждого дропа.
- Интеграция отзывов пользователей с системой репутации (Reputation System) для стимулирования активности.
